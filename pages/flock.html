screenshot of live site

Flock is a web application that allows users to search the twitter API, filter based on various parameters, and download tweets. This project was created for the communications department at Northeastern University. I was responsible for the design of the application, and shared development with two other students.

While doing research, students and professors would manually search the Twitter website for tweets, then enter them into a spreadsheet with the any data they could gather. Our tool allows them tool search for tweets based on their desired parameters, such as location, date, and keywords, and easily looks through the results. Tweets could then be downloaded into a CSV file that included all of the tweets' metadata and is easily imported to Excel.
